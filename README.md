# moonscrobbler
a discord bot that posts the songs you've been listening to, via lastfm scrobbles
created by moon

## setup instructions

### step 1: set your variables!

- LASTFM_USER (your lastfm username)
- API_KEY (your lastfm api key)
- DISCORD_TOKEN (your discord bot token)
- CHANNEL_ID (the discord channel id to post the songs to)

### step 2: that's literally it
